App Guide & Features Explained
▾
GPS Status Bar
STATUS
LOCKED = good signal. WEAK = marginal. POOR = don't start a run yet.
ACCURACY
How far off the GPS could be in metres. Under 10m is ideal for timing runs.
FIX AGE
Seconds since the last GPS update. Under 1.5s is green. Over 3s means signal is struggling.
SIGNAL BARS
Visual GPS strength. Four green bars = best possible lock.
Speedometer
GAUGE
Analog needle driven by GPS speed. Tap MPH ⇄ to toggle between mph and km/h.
GREEN MARKER
Your target speed marked on the arc — see exactly how far you have to go.
BLUE MARKERS
Your split speed checkpoints marked on the gauge arc.
Acceleration Mode
HOW IT WORKS
Set a target speed, tap ARM, then go. Timer starts automatically when you move and stops when you hit the target.
TARGET SPEED
The speed you want to time to. Classic run is 0–60 mph.
SPLITS
Optional intermediate speeds e.g. 30, 45. Split times are recorded automatically as you pass each one.
Braking Mode
HOW IT WORKS
Set your entry speed, ARM, accelerate to that speed, then brake hard. Timer runs until you stop.
FROM SPEED
The speed you'll be doing when you start braking. Timer kicks in the moment you hit it.
Run Controls
ARM
Puts the app on standby. Nothing is recorded until the run actually starts.
RESET
Cancels the current run and returns to idle.
ARM AGAIN
On the result screen — instantly re-arms for another back-to-back run.
Results & History
BEST VS LAST
Compares your most recent run against your personal best for the same run type.
RUN HISTORY
Every run saved automatically. Tap any run to expand split details. Gold highlight = personal best.
NEW BEST
Badge shown on the result popup and in history when you beat your previous best time.
